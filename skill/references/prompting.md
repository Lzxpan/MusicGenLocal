# Prompting

Use short, concrete prompts first. Prefer 4-12 second test renders while iterating, then render 30 seconds only after the prompt is stable.

For loopable game music, include:

- `instrumental only`
- `seamless loop`
- `no vocals`
- `no spoken words`
- `no dramatic intro`
- `no fade-out outro`
- BPM or pacing
- core instrument palette
- game context

Example prompts:

- `Retro arcade puzzle game background music, instrumental only, seamless loop, no vocals, no spoken words, no dramatic intro, no fade-out outro, steady 8-bit synth pulse, bright focused energy, 125 BPM.`
- `Calm lofi puzzle game loop, instrumental only, seamless loop, no vocals, soft electric piano, light drums, warm bass, no fade-out outro.`

Avoid broad requests like `make cool music` because they reduce control over loopability and texture.
